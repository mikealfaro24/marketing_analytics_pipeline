📊 What the Documentation Includes

The documentation walks through the full development process of the analytics project, including:

• Project goals and business questions
• Data sources and structure
• Data pipeline design
• Bronze, Silver, and Gold data layers
• Data cleaning and transformation logic
• SQL analysis queries
• Dashboard metrics and KPI definitions
• Insights and observations from the analysis

The goal is to show not only the final outputs, but also how the project was designed and structured from start to finish.

⚙️ Why I Use Notion for Documentation

For larger projects, I prefer documenting the workflow in Notion because it allows me to:

• Organize ideas and project structure clearly
• Track the different stages of development
• Document decisions and assumptions
• Keep notes alongside diagrams, queries, and analysis steps

This helps create a clearer narrative of the project beyond the code itself.
